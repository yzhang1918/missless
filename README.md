# missless

From FOMO to Focus.

`missless` is an AI judgment engine for links and ideas. This repository is the single source of truth for product intent, architecture decisions, and implementation standards.

## Project Stage

- Current phase: POC/MVP (self-host first)
- Product mode: CLI-first ingestion and review workflow
- Source-of-truth policy: repository-local docs, specs, and plans only

- Repository map: [ARCHITECTURE.md](./ARCHITECTURE.md)

## Core Promise

Drop a link. Get a decision.

The system turns sources into:
- concise claims (Atoms)
- structured outputs (Artifacts)
- auditable evidence links (Segments)
- explainable read-priority ratings (`skip`, `skim`, `read`, `deep_read`)
