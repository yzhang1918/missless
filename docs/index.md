# Documentation Index

Status: Active

This is the canonical entry point for repository knowledge.

## How to Use This Index

- Start from the catalog below.
- Use `Status` to avoid relying on deprecated documents.
- Use `Scope` and `Summary` to find the smallest relevant context first.
- To answer "what should we do next?", start from `Execution Plans -> Kanban`.

## Status Legend

- `Active`: current source of truth
- `Draft`: in-progress proposal
- `Deprecated`: historical only, replaced elsewhere
