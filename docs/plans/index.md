# Plans Index

Status: Active

## Purpose

Provide one clear place to navigate the tracked product plans and archived
execution history for `missless`.

## How To Use This Index

- Run `harness status` first when you need the current execution state.
- Use `docs/plans/active/` for tracked plans that are still in progress.
- Use `docs/plans/archived/` for durable product execution history.
- The generic plan lifecycle, approval, archive, and land workflow is owned by
  `easyharness`, not a separate repository-local plan contract.

## Status Legend

- `Active`: currently used planning asset
- `Draft`: under discussion
- `Superseded`: replaced by a newer planning asset
