# Execution Plans Index

Status: Active

## Purpose

Provide a single tactical operating spine for "what next", active execution, and unresolved follow-ups.

## Status Legend

- `Active`: currently used planning assets
- `Draft`: template/proposal not yet adopted
- `Deprecated`: retained for history

## First Stop

If you ask "what should we do next?":
1. open [Kanban](./kanban.md)
2. open [Backlog](./backlog.md)
3. pick or create an active plan in `active/`
