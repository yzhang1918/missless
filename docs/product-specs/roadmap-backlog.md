# Roadmap Backlog (Beyond POC)

Status: Active

## Deferred Product Experience

- Web UI for browsing, comparison, evidence highlighting, bulk review
- Mobile clients and mini-programs
- Inbox/triage reminders and recurring workflows
- Voice-first capture and transcription

## Deferred Knowledge Organization

- Multi-vault/project isolation
- Automated classification and AI-suggested taxonomy
- Knowledge cards and materialized views
- User mastery state modeling (`unknown`, `aware`, `understand`, `apply`)
- Debate trees and position aggregation

## Deferred Connectors

- YouTube and podcast ingestion with timestamp evidence
- Twitter/X thread parsing
- RSS subscription and auto-ingest
- Batch local file ingestion (PDF/Markdown/EPUB)
- Structured extraction from figures/charts

## Deferred Scoring Intelligence

- Long-term channel/author quality models
- Personalized ranking from feedback and behavior
- Better time-cost estimation tied to user reading patterns

## Deferred Learning Flywheel

- Prompt optimization from user merge/contradiction feedback
- Classifier/reranker training for equivalence and contradiction
- Active learning for uncertain alignment decisions
- Online weight updates for scoring policy

## Deferred Platformization

- MCP server and CLI tooling standardization
- Extension registry and plugin ecosystem
- Skill packaging/versioning

## Deferred Compliance and Business

- Multi-tenant SaaS, billing, quota, cost control
- Regulatory and app-store compliance execution
- Privacy and data sovereignty hardening
