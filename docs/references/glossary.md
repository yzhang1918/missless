# Glossary

Status: Active

- **Source**: A single ingested content unit (URL, paper, file, etc.).
- **Segment**: A traceable fragment of a source with locator metadata.
- **Atom**: A short, scan-friendly claim or insight.
- **Artifact**: Structured output object extracted from a source.
- **Alignment**: Mapping extracted items to existing knowledge (`new`, `duplicate_of`, `equivalent_to`, `qualifies`, etc.).
- **Ingest Run**: One full processing unit from fetch to commit-ready proposal.
- **Proposal**: Reviewable result package before commit.
- **Commit Plan**: The resolved set of writes after alignment and human review.
- **Slop**: Low-signal, repetitive, or derivative content with little incremental value.
