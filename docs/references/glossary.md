# Glossary

Status: Active

- **Source**: A single ingested content unit (URL, paper, file, etc.).
- **Evidence Anchor**: A traceable reference back to a concrete part of a source.
- **Atom**: A short, scan-friendly claim or insight.
- **Artifact**: A structured output object extracted from a source.
- **Alignment**: Comparing extracted outputs with existing knowledge and deciding their relationship.
- **Ingest Run**: One full processing unit from fetch to proposal/commit.
- **Proposal**: Reviewable result package before commit.
- **Commit Plan**: The resolved set of writes after alignment and human review.
- **Slop**: Low-signal, repetitive, or derivative content with little incremental value.
